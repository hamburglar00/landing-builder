## Landing Builder – Next.js + TypeScript + Tailwind

Constructor de landing de plantilla fija, con login mock, dashboard con editor y preview en vivo.

### Requisitos

- Node.js 18+
- npm, pnpm o yarn

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Luego abre `http://localhost:3000`.
